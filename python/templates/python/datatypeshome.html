1. Numeric Data Types in Python 
Numeric data types are used to store number-based values in Python.
These are mainly used for calculations, measurements, counters, mathematical formulas, and scientific computations.
Python provides three main numeric types:
    
Integer
Floating Point
Complex
a) Integers

An integer is used to store whole numbers.
These numbers do not contain any decimal value.
Examples:
x = 10
y = -25
z = 0
Integers can be:  
positive
negative
zero
They are mostly used in:
counting
indexing
calculations
loops
Example:
age = 18
marks = 95
Python integers support arithmetic operations like +, -, *, /, %.
b) Float

Float stores decimal numbers / floating point values.
A float always contains a decimal point.
Even though it looks like a whole number, because of the decimal it is a float.
Used in:

Percentage
Marks Average
Measurements
Money Vakues
Scientific Calculations

Floats are generally stored using double precision.
c) Complex

Complex numbers are used for advanced mathematics and engineering.
They are written in the form:

a + bj

Where:

a = real part
b = imaginary part
j = square root of -1
2. Boolean

Boolean stores only two values:

True
False

It is mainly used in:

conditions
loops
comparisons
logical statements

Boolean is extremely important for if, while, and comparisons.
3. Dictionary

Dictionary stores data in key-value pairs.

Example:

student = {
    "name": "Harshit",
    "age": 18
}

Here:

name → key
Harshit → value

Dictionaries are mutable.

Very useful for storing structured data.

4. Set

Set stores unique values only.

Example:

s = {1, 2, 3, 3}

Output:

{1, 2, 3}

Duplicate values are removed automatically.

Sets are unordered.

Useful for unique items and mathematical operations.

5. Sequence 
Sequence means ordered collection of values.

Python sequence types include:

string
list
tuple

These store multiple values in order.

a) List

List stores multiple values in ordered form.

Example:

nums = [1, 2, 3, 4]

Lists are mutable, meaning values can be changed.

Example:

nums[0] = 100

List can store mixed types:

data = [10, "python", True]
Click Here To Study Methods Of Lists

b) Tuple

Tuple is similar to list but immutable.

Example:

point = (10, 20)

Once created, values cannot be changed.

Used when data should remain fixed.

Example:

days = ("Mon", "Tue", "Wed")
Click Here To Study Methods Of Tuples

c) String

String is used to store text / characters.

Example:

name = "Harshit"

Strings are written inside:

single quotes ' '
double quotes " "

Example:

city = "Moga"

Strings are immutable, meaning once created, characters cannot be changed directly.
