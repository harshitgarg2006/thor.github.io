1. While Loop 

A while loop runs as long as a given condition remains true.

This means Python first checks the condition:

if true → loop runs
if false → loop stops

The condition is checked before every iteration.

This loop is best used when the number of repetitions is not known in advance.

Examples of where it is useful:

keep asking for password until correct
keep taking input until user enters 0
game loop
menu-driven programs
2. Nested While Loop 

A nested while loop means a while loop inside another while loop.

The outer loop runs first.

For every single iteration of the outer loop, the inner loop runs completely.

This is mainly used in:

matrix problems
tables
pattern printing
coordinate-based problems
3. Range Function

The range() function is mainly used with for loops.

It generates a sequence of numbers.

Syntax:

range(start, stop, step)

Where:

start → starting value
stop → ending limit (not included)
step → increment / decrement
4. For Loop

A for loop is used to iterate over a sequence or range.

This loop is best when the number of iterations is known.

It automatically handles the loop variable and movement.

It is simpler than while for fixed repetitions.

Used for:

counting
list traversal
string traversal
patterns
tables
5. Nested For Loop

A for loop inside another for loop.

The inner loop runs fully for each outer loop iteration.

This is extremely important in:

pattern printing
2D arrays
matrix traversal
multiplication tables
combinations
6. Infinite Loop 

An infinite loop is a loop that continues to run forever because its stopping condition never becomes false.
This usually happens when the condition in a loop always remains true or when the variable controlling the loop is not updated properly.
Infinite loops are commonly associated with while loops, but they can occur in other looping structures as well.
While unintentional infinite loops can cause programs to freeze or consume system resources, intentional infinite loops are sometimes used in applications like game engines, servers, and continuous monitoring systems.
Proper control conditions are important to avoid unwanted infinite execution.
